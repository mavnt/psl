# psl

Multiple prints on the same line.

[![asciicast](https://asciinema.org/a/TC204YaZR092nw6oTmj1cAN2F.svg)](https://asciinema.org/a/TC204YaZR092nw6oTmj1cAN2F)

```python
items = [
    "Honujwa",
    "Ijicudik",
    "Beparhu",
    "Loewavac",
    "Tafjoiwo",
    "Gamufiolu",
    "Jagsajuno",
    "Wadigco",
    "Umpepit",
    "Kojsijneb",
    "Wuhfona",
    "Culehlo",
    "Juuwuvih",
]

for item in items:
    psl(f"Processing {item}...")
    time.sleep(0.3)
print()
```

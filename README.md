# psl

Multiple prints on the same line.

```python
items = [
    "Honujwa",
    "Ijicudik",
    "Beparhu",
    "Loewavac",
    "Tafjoiwo",
    "Gamufiolu",
    "Jagsajuno",
    "Wadigco",
    "Umpepit",
    "Kojsijneb",
    "Wuhfona",
    "Culehlo",
    "Juuwuvih",
]

for item in items:
    psl(f"Processing {item}...")
    time.sleep(0.3)
print()
```
